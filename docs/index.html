FluidDial-CYD

Web Installer for the CYD FluidNC Pendant

Requirements

  • Google Chrome or Microsoft Edge (version 89+)
  • CYD (Cheap Yellow Display) ESP32 board
  • USB data cable (not charge-only)
  • Windows, macOS, or Linux
Web Serial is not supported in Firefox or Safari. Use Chrome or Edge.

Install Firmware

v1.6.0 — Latest Release

Connect your CYD to USB, then click the button below.
Select the correct COM port when prompted.

Your browser does not support Web Serial.
Please use Chrome or Edge.

First install: You will be prompted to erase the device — select Yes to ensure a clean installation.

Update: If you have already installed FluidDial-CYD, you can update without erasing — your settings will be preserved.

After Installing

First boot — screen detection: the pendant will display "Capacitive — Tap the Screen" then "Resistive — Tap the Screen". Tap when your screen type is shown. The result is saved and the pendant reboots automatically — this only happens once.

Connect your CYD pendant to your FluidNC controller via UART (TX/RX). The pendant will automatically detect the controller and display axis positions, machine state, and allow jogging, homing, probing, and spindle control.

See the GitHub repository for full documentation and wiring instructions.
